Лаба 4.

Нужно взять планету по вариантам:
1. Меркурий
2. Венера
3. Марс
4. Юпитер
5. Сатурн
6. Уран
7. Нептун
8. Плутон

Взять её орбитальные элементы из системы HORIZONS за прошедший год (напоминаю, что для каждой планеты такое понятие как год разное).

Орбитальные элементы нужно пересчитать в декартовые одним из 3х способов:
1. Формулы в лекции
2. Формулы в лабе (лабораторная работа №3 из перечня, файл прилагается)
3. По ссылочке из ESA: https://gssc.esa.int/navipedia/index.php/Osculating_Elements

Теперь нужно нарисовать 2 орбиты:
1. Орбита по всему набору x, y, z. Будет выглядеть волнисто, как во 2ой лабе.
2. Орбита по x, y, z, vx, vy, vz для начальной точки. Будет выглядеть как эллипс. Сам процесс рисования можно стянуть из 3ей лабы нашей.
